1. install chokidar using the command "npm i chokidar"

terminal

copy 
npm i chokidar

2. downlload all the necessary files from here

3. place all the files inside the zip next to index.html (in the main directory)

4. add this to package.json:

package.json

copy 
"scripts": {
 "powertik": "node watch-pt.js"
},

5. correct the urls inside: powertik-files.js

powertik-files.js

copy 
const css = "./styles/main.css";
const index = "./index.html";

6. run powertik framework by runing this command in your terminal:

terminal

copy 
npm run powertik

7. now you can link main.css to your html file

index.html

copy 
< link rel="stylesheet" href="./styles/main.css" />

7. now you can link main.css to your html file
