How to use Dark Mode:

Writing Dark mode class is simple, write "dark:" + your class like this:
  -  dark:{class}
and add a button to toggle between dark mode and light mode:
  -  < button id="darkModeBtn"> dark mode < / button>
then add this code to your javascript file:

javascript

copy 
let allDarkModes = document.querySelectorAll("[class*='dark:']");
let darkModeBtn = document.getElementById("darkModeBtn");
let darkMode = false;
darkModeBtn.addEventListener("click", activateDarkMode);
function activateDarkMode() {
  darkMode = darkMode ? false : true;
  allDarkModes.forEach((elem) => {
    darkMode
      ? (elem.className = elem.className.replaceAll("dark:", "pt-dark-"))
      : (elem.className = elem.className.replaceAll("pt-dark-", "dark:"));
  });
}

How to add custom css classes:

You can add css shortcuts in "powertik-files.js" in this format:
  -  ["className","property","property value"],
  -  ["blue","color","#0000FF"],

powertik-files.js

copy 
            const custom_shortcuts = [
    ["blue","color","#0000FF"],
]

How to use gradient backgrounds:

You can use gradient like this:
  -  gradient-[0deg,$purple-00_50%,blue_50%]

html

copy 
            class = "gradient-[90deg,$lightblue-200,$purple-00]"

How to add hover on elements:

You can add hover effect on elements using "hover:" like this:
  -  hover:{class}
  -  hover:color-[$lightblue-200]

html

copy 
            class = "hover:color-[$lightblue-200]"
